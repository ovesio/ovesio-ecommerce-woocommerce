== Description ==

**Ovesio - Ecommerce Intelligence** plugin empowers your WooCommerce store with advanced AI-driven insights, stock management forecasting, and strategic consulting. This plugin generates secure data feeds for your products and orders, allowing the Ovesio platform to analyze your business performance in real-time.

== Installation ==

1.  **Edit Settings:** Navigate to **WooCommerce** > **Ovesio Ecommerce Intelligence** in the main menu.
2.  **General Settings:**
    *   **Status:** Set to **Enabled**.
    *   **Order Export Period:** Choose the historical period for analysis (Last 12 Months or Last 24 Months).
    *   **Order Statuses:** Select the order statuses to export (e.g., Completed, Processing). Leave empty to use default valid orders.
3.  **Data Sources:**
    *   The plugin automatically generates secure **Product Feed** and **Order Feed** URLs displayed on the configuration page.
    *   Copy these URLs.
4.  **Connect to Ovesio:**
    *   Create an account at [ovesio.com](https://ovesio.com).
    *   Navigate to the **Data Sources** section in your Ovesio dashboard.
    *   Paste the **Product Feed URL** and **Order Feed URL** generated by this plugin.
5.  **Save:** Click the **Save Changes** button.

== Frequently Asked Questions ==

= Important Note on Security =

When you install this plugin, a unique **Security Hash** is automatically generated. This hash protects your data feeds from unauthorized access.

*   **If you Uninstall and Reinstall** the plugin, a **NEW** hash may be generated or you might need to reset it.
*   This will change your feed URLs.
*   You must update the URLs in your Ovesio.com account whenever the hash changes to maintain the connection.
